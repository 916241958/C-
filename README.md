# C++ Variadic Template Thread Pool

## 简介

这个项目实现了一个基于C++11及以上版本的可变参模板线程池。它允许用户方便地创建线程池，并向其中提交多种不同参数的任务。

## 特性

- **快速部署**：只需包含头文件即可使用线程池。
- **易于使用**：提供简单直观的API接口。
- **高性能**：优化的线程管理和任务调度，以提高并发执行效率。
- **灵活性**：支持可变参数模板，可以接受不同数量和类型的参数。
